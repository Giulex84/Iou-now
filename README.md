# IOU

Applicazione semplice e veloce per tracciare debiti e crediti.

## Funzionalità
- Aggiungi IOU (Mi devono / Devo)
- Storico IOU
- Totali automatici in home
- Eliminazione IOU
- Grafica moderna e uniforme

## Tecnologie
- Next.js 15
- TailwindCSS
- Supabase

## Struttura
- `/app` – pagine Next.js
- `/lib` – logica IOU + supabase
- `/public` – assets
- `/components` – componenti UI

## Deploy
Automatico tramite Vercel.
